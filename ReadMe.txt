프로젝트명: 주가 예측
개발 기간: 2019. 07. 24 (수) ~  2019. 08. 02 (금)[10일]
개발 목적: LSTM으로 주가를 예측해보자
개발자: 상병 이현
사용 기술: LSTM, Momentum Optimizer
개발 언어: HTML, Javascript

배운점:
 - 외부 요인에 의해 변하는 값은 신경망으로 예측할 수 없다.
   = 종속 변수가 있어야 값을 예측할 수 있다.
   → 과거 주가는 오늘 주가와 독립 변수이다. 그래서 예측이 안된다.
 - LSTM을 개발해봤다.
 - Continuous version(forward 할 때 이전 forward에서 prevH, prevC를 가져오는 버전)이 성능이 더 좋다.
   → Non-Continuous version만의 이점도 있다.

보완할점:
 - 하고자 한다면, 주가의 종속 변수를 찾아야 한다.
 - 배열끼리의 연산 함수를 만드니 LSTM을 개발하기 쉬웠다. 행렬 연산을 하는 함수를 만들면 더 깔끔해 보일거 같다.
